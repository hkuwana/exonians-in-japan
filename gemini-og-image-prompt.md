# Gemini Image Generation Prompt for Open Graph Image

Use this prompt with Gemini (or similar AI image generator) to create an Open Graph preview image for the Exonians in Japan website.

## Prompt

```
Create a professional, clean Open Graph social media preview image (1200x630 pixels) for "Exonians in Japan" - a Phillips Exeter Academy alumni community in Japan.

Design requirements:
- Background: Deep crimson red gradient (#9B2335 to #7a1c2a), similar to Exeter's school colors
- Main text: "Exonians in Japan" in large, elegant white serif font (like Crimson Text or Georgia)
- Subtitle: "Phillips Exeter Academy Alumni Network" in smaller white sans-serif
- Visual elements: Subtle Japanese-inspired design elements like:
  - A minimalist torii gate silhouette
  - Cherry blossom branches
  - Or Mount Fuji outline
- Keep it professional and sophisticated, not cartoonish
- Include "Non Sibi" motto subtly in gold (#C4A35A) color
- Leave space on the right or bottom for visual breathing room

Style: Modern, professional, clean. Think alumni association meets elegant Japanese aesthetics.
```

## Alternative Prompts

### More Minimalist Version
```
Create a minimalist Open Graph image (1200x630px) with:
- Solid crimson background (#9B2335)
- Large white "E" letter in serif font (representing Exeter)
- Small white torii gate icon below
- "Exonians in Japan" text
- Clean, modern, professional look
```

### Photography-Based Version
```
Create an Open Graph image (1200x630px) featuring:
- Background: Blurred photo of Tokyo skyline at dusk
- Crimson (#9B2335) gradient overlay at 60% opacity
- White text "Exonians in Japan" centered
- Subtitle "Connect with PEA Alumni across Japan"
- Professional, welcoming feel
```

## After Generation

1. Save the image as `og-image.png` (or `og-image.jpg`)
2. Place it in the root directory of the project
3. The meta tags in `index.html` are already set up to use it

## Image Specifications

- **Size**: 1200 x 630 pixels (Facebook/LinkedIn optimal)
- **Format**: PNG or JPG
- **File size**: Keep under 1MB for fast loading
- **Safe zone**: Keep important content away from edges (50px margin)
